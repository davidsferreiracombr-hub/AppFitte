# **App Name**: Fit Sweet Delights

## Core Features:

- Recipe Database: Store and manage over 500 healthy dessert recipes.
- Detailed Recipe View: Display each recipe with complete details, including ingredients, preparation steps, cooking time, and nutritional information.
- Recipe Search and Filtering: Allow users to search for recipes based on keywords, dietary restrictions (e.g., gluten-free, dairy-free), and nutritional content.
- AI Recipe Recommendation: Suggest recipes to the user based on their stated dietary goals, allergies, preferred flavors, and past recipe interactions. The AI will act as a 'tool' in filtering and ordering results based on inferred preferences.
- User Profiles: Allow users to create profiles to save their favorite recipes, track their dietary intake, and set personal fitness goals.
- Nutrition Information API Integration: Automatically fetch and display comprehensive nutritional data for each recipe using an external nutrition API. 
- Community Forum: Enable users to share their experiences, modifications to recipes, and tips for healthy eating through a community forum.

## Style Guidelines:

- Primary color: Soft coral (#F08080), evoking warmth and health.
- Background color: Pale peach (#F5DDC4), providing a light, clean backdrop.
- Accent color: Muted rose (#D46A6A) for key actions and highlights.
- Body font: 'PT Sans', sans-serif, for a modern, approachable feel.
- Headline font: 'Alegreya', serif, for a touch of elegance and readability.
- Use simple, minimalist icons related to food, fitness, and health to enhance visual appeal and usability.
- Ensure a clean, organized layout with plenty of white space to emphasize simplicity and readability.
- Use subtle, smooth animations when loading recipes and transitioning between sections to enhance the user experience.
